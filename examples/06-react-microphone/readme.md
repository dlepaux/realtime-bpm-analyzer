# React Microphone Example

This example demonstrates real-time BPM detection from microphone input using React.

## Features

- Real-time BPM analysis from microphone
- Uses `createRealtimeBpmAnalyzer()`
- Proper microphone permission handling
- Start/stop recording controls
- React hooks for state and lifecycle management

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3005](http://localhost:3005) in your browser.
