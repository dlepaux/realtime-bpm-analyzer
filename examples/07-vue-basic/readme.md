# Vue Example

Vue 3 integration with Composition API and TypeScript for offline BPM analysis.

## Features

- Vue 3 with Composition API
- `<script setup>` syntax
- Offline analysis with `analyzeFullBuffer`
- TypeScript support
- Reactive state management
- Proper lifecycle management with `onMounted` and `onUnmounted`

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3006](http://localhost:3006) in your browser.
