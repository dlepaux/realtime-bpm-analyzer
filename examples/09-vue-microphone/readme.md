# Vue Microphone Example

This example demonstrates real-time BPM detection from microphone input using Vue 3.

## Features

- Real-time BPM analysis from microphone
- Uses `createRealtimeBpmAnalyzer()`
- Proper microphone permission handling
- Start/stop recording controls
- Vue 3 Composition API with `<script setup>`

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3008](http://localhost:3008) in your browser.
